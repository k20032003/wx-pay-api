WeiXin Pay API v3 for PHP, Modified by Goldeagle

## 这是什么？
这个是基于微信支付官方 PHP SDK V3 基础上修改的一套代码，实现了几个目的：
* 尽量兼容 PSR-2；
* 保证对 PHP 5.6 以上版本的支持（例如：用 php://input 替代 $GLOBALS['HTTP_RAW_POST_DATA']）；
* 修正了官方 SDK 中的一些错误。

## 还有什么事情要做？
* 完善对 composer 的支持，支持 PSR-4；
* 提供并完善样例代码；
* 提供并完善说明文档，起码要比官方文档更详尽。
